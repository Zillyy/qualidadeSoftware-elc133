# Projeto de Testes usando Junit

Projeto desenvolvido para a disciplina de Qualidade de Software.

## Começando

As instruções a seguir irão ajudar a rodar o jogo em seu computador.
*Obs: Para rodar os TESTES, não é necessário fazer estas configurações.

### Pré-requestiso para **rodar o jogo na sua máquina**

* Mudar o JDK para a versão 32 bits. 

* Clique com o botão direito no seu projeto > Propriedades > Executar > Opções de VM.  
Aqui você deve inserir o caminho das DLLs do projeto usadas pela LWJGL no Windows.  
As DLLs encontra-se na pasta: lib/win32. Por exemplo,  *Djava.library.path="F:\qualidadeSoftware-elc133\diamondCrushRun\lib\win32"*


## Rodando os testes

Para rodar os testes, não é necessário modificar nada dentro do projeto ou na sua máquina.  
Basta abrir o arquivo de teste que deseja executar e pressionar <kbd>shift</kbd>+<kbd>F6</kbd>  
Ou clicar com o botão direito dentro do arquivo e escolher a opção *Executa Arquivo*.
